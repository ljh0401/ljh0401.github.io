# 이진 탐색 알고리즘

이진 탐색은 정렬된 배열에서 특정 값을 찾는 알고리즘입니다. 시간 복잡도는 O(log n)으로, 선형 탐색의 O(n)보다 훨씬 효율적입니다.

## 알고리즘 원리

1. 배열의 중간 값을 확인합니다.
2. 찾고자 하는 값이 중간 값보다 작으면 왼쪽 부분을, 크면 오른쪽 부분을 탐색합니다.
3. 찾을 때까지 이 과정을 반복합니다.
